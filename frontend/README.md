# 🎨 ShadowSend Frontend

A premium React-based interface for the ShadowSend DApp on Midnight Network.

## 🛠️ Tech Stack
- **Framework**: Vite + React
- **Styling**: Tailwind CSS + Framer Motion
- **Wallet**: Midnight Wallet / mnLace
- **State**: RxJS for real-time wallet polling

## 🚀 Development
```bash
yarn install
yarn dev --port 3001
```

## 🔐 Privacy Features
- **Lace Connector**: Real-time shielded/unshielded balance tracking.
- **ZK-Proof Integration**: Logic ready to connect with `shadowsend-contracts` using `midnight-js`.
- **Zero Horizontal Overflow**: Fully responsive grid engine for multi-recipient rows.
